# Sudoku

Sudoku es un juego combinatorio de colocación de números basado en la lógica. En el sudoku clásico, el objetivo es llenar una cuadrícula de 9 × 9 con dígitos para que cada columna, cada fila y cada una de las nueve subcuadrículas de 3 × 3 que componen el tablero contenga todos los dígitos del 1 al 9. Este juego proporciona una cuadrícula parcialmente completa, que en la gran mayoria debe tener una única solución.

#### Reglas del sudoku

1. Solamente puedes usar numeros del 1-9.
2. No puedes usar el mismo numero mas de una vez en una fila/columna/3x3.


### Como navegar a travez del menu principal y empezar un nuevo juego

- Elige una dificultad presionando las flechas.
- Haz click en "Juego Nuevo" para empezar una nueva partida con la dificultad elegida.

### Como jugar Sudoku

 - Click en una celda vacia y typear un numero usando el teclado.
 - Para volver atras, presiona el boton "Volver".
 - Si crees que es necesario, puedes resetear la partida actual presionando dicho boton.
 - Muy cansado o no encuentras la solucion? Prueba pulsando el boton de "Solucion Automatica!"
